** <span> -----> it is used to style the text even in a paragraph.
** <strong> -----> it is used to apply bold to a specific text.
** <u> -----> it is used to style the specific word or text.
** <!doctype html> ---> it is used to provide the latest features of html to
    be applicable to the webpage. if not the previous features are used for it.
** <body> ----> that contains all the elements on the webpage.
** void elements -----> these are which that contains just theopening tag but no 
    closing tag.
** In order to get different types of fonts, visit google fonts and download the code
    from it and put in head tag.
** In html there are three types of elements:-
    -->> block element - it takes the entire line. eg:- <p></p>
    -->> inline block element - it takes only as much space needed. eg:- <img>, <input>
    -->> inline element - it appears only within a line of text.
** By default <button> is a kind of inline element.
